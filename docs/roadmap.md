# Roadmap

## Alpha 1: Web/Windows Editor Foundation

- Build a usable single-scene editor with SVG preview.
- Support shape/text layers, transforms, opacity, styling, blur/shadow effects.
- Add keyframes, easing, playback, scrubbing, and undoable patches.
- Add AI dock for draft, edit, explain, and automate operations.
- Export project JSON, SVG snapshot, and Lottie subset with warnings.

## Alpha 2: Serious Motion Controls

- Nested scenes and exposed properties.
- Masks, mattes, and richer blend modes.
- Better path editing and SVG import fidelity.
- Audio tracks with waveform display and sync markers.
- Graph editor with editable Bezier handles.

## Beta: Production Output

- PNG sequence, GIF, WebM, and MP4 export.
- dotLottie package export.
- Template-safe exposed controls.
- Secure AI relay and managed/BYO-key tiers.

## Native Track

- Shared document model package.
- Metal renderer for macOS/iPadOS.
- Pencil-first canvas editing and touch timeline.
- Local preview parity tests against web renderer.
